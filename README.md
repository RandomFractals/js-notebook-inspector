# Dev Build

```bash
$ git clone https://github.com/RandomFractals/js-notebook-inspector
$ cd js-notebook-inspector
$ npm install
$ code .
```
`F5` to launch JS Notebook 📓 Inspector 🕵️ extension VSCode debug session.
